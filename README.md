個人利用・商用利用ともに無料でご利用いただけます。

## デモサイト
https://thai-massage.pages.dev/

## 特徴

- レスポンシブデザイン対応
- SEO対策済み
- 軽量設計
- カスタマイズしやすい構成
- MITライセンス

## 技術スタック

- React
- Vite
- TypeScript
- Tailwind CSS

## 使い方

1. リポジトリをクローン

git clone https://github.com/H2works/thai-massage

2. 依存関係をインストール

npm install

3. 開発サーバーを起動

npm run dev

## ビルド方法

npm run build
